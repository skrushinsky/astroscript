NAME

AstroScript::Nutation - nutation and obliquity of ecliptic

VERSION

Version 0.01

SYNOPSIS

    use AstroScript::Nutation qw/:all/;

    $delta_psi = nut_lon($t); # nutation in longitude
    $delta_eps = nut_obl($t); # nutation in ecliptic obliquity
    $epsilon = ecl_obl($t); # obliquity of ecliptic
    ...

EXPORT

  • "$delta_psi = nut_lon($t)"
  • "$delta_eps = nut_obl($t)"
  • "$epsilon = ecl_obl($t)"

SUBROUTINES/METHODS

$delta_psi = nut_lon($t)

Nutation in longitude (radians)

$delta_eps = nut_obl($t)

Nutation in obliquity (radians)

$epsilon = ecl_obl($t)

Obliquity of ecliptic in radians. Accuracy is 0.01" between 1000 and 3000, and a few arc-seconds after 10,000 years.
