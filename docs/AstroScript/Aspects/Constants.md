# NAME

AstroScript::Aspects::Constants

# SYNOPSIS

    use AstroScript::Aspects::Constants qw/:types :influences/;

# DESCRIPTION

Constants used by aspects-related modules.

# EXPORTED CONSTANTS

## Influences

- `$POSITIVE`
- `$NEGATIVE`
- `$NEUTRAL`
- `$CREATIVE`

## Aspect Types

- `$MAJOR`
- `$MINOR`
- `$KEPLER`
