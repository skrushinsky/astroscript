NAME

AstroScript::Aspects::Aspect - Base class for an aspect.

SYNOPSIS

  package AstroScript::Aspects::Aspect::Mercury;
  use base qw/AstroScript::Aspects::Aspect/;
  ...

  sub  heliocentric {
    # implement the method
  }

DESCRIPTION

Base class for an aspect. In most cases there is no need to use this class, unless you wich to use some rare experimental aspect. All the standard aspects are created and exported by AstroScript::Aspects module.

METHODS

AstroScript::Aspects::Aspect->new( %options )

Constructor.

Named arguments:

  • name — the name
  • brief_name — 3-letters identificator for tables
  • value — arc-distance in degrees, e.g. 120 for trine.
  • influence — one of constants exported by AstroScript::Aspects::Constants: $POSITIVE, $NEGATIVE, $NEUTRAL, $CREATIVE.
  • type — one of constants exported by AstroScript::Aspects::Constants: $MAJOR, $MINOR, $KEPLER.

$self->name

Read-only accessor for aspect's name

$self->brief_name

Read-only accessor for 3-letters identificator

$self->value

arc-distance in degrees

$self->influence

one of constants exported by AstroScript::Aspects::Constants: $POSITIVE, $NEGATIVE, $NEUTRAL, $CREATIVE.

$self->type

one of constants exported by AstroScript::Aspects::Constants: $MAJOR, $MINOR, $KEPLER.
