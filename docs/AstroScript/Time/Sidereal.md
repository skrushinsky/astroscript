# SUBROUTINES/METHODS

## ramc($jd, $lambda)

Right Ascension of the Meridian

### Arguments

- **$jd** — Standard Julian Date.
- **$lambda** — geographic longitude in degrees, negative for East

### Returns

Right Ascension of Meridian, arc-degrees
